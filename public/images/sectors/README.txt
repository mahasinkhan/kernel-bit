Sector imagery
==============
Drop six JPGs in this folder, using exactly these names:

  finance.jpg
  healthcare.jpg
  public-sector.jpg
  energy.jpg
  education.jpg
  retail.jpg

About 1600 x 1000, under 300 KB each. They are auto-toned to the blueprint
palette in CSS, so ordinary stock photos come out looking like a set.
If a file is missing the card falls back to a blue plate - nothing breaks.
