Photography
===========
Drop these eight files in place. Any decent photo works: everything is
auto-toned to the porcelain / ultramarine palette in CSS, so a mixed set
still comes out looking like one brand system.

  /public/images/hero.jpg                    1400 x 1700  (portrait)
  /public/images/proof.jpg                   1200 x 1500  (portrait)
  /public/images/sectors/finance.jpg         1600 x 1000
  /public/images/sectors/healthcare.jpg      1600 x 1000
  /public/images/sectors/public-sector.jpg   1600 x 1000
  /public/images/sectors/energy.jpg          1600 x 1000
  /public/images/sectors/education.jpg       1600 x 1000
  /public/images/sectors/retail.jpg          1600 x 1000

Keep each one under about 300 KB. If a file is missing, the card falls back
to a blue plate - nothing breaks, nothing looks unfinished.

Good free sources: unsplash.com, pexels.com. Prefer real work: people at
screens, server halls, clinical settings, substations. Avoid stock handshakes.
